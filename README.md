# AI Skin Lesion Triage Assistant

## 📌 Project Description
This is a full-stack AI web application that allows users to upload skin lesion images and receive a prediction indicating potential risk levels such as:

- Benign (Low Risk)
- Malignant (High Risk)
- Needs Further Medical Check

The system demonstrates the integration of frontend, backend, and AI components in a real-world application.

---

## 🛠️ Technologies Used

### Frontend
- React.js
- Axios

### Backend
- Node.js
- Express.js
- Multer (file upload)
- bcrypt.js (authentication)
- JWT (JSON Web Tokens)

### AI Model
- Python (predict.py)
- Random-based prediction (simulation)

---

## 🚀 Features
- Upload skin lesion image
- Image preview before upload
- AI-based prediction results
- User registration and login
- Dashboard with logout functionality
